# 标准站姿角色参考图 Prompt

## 参考图用途

- 图片 1（`全身.webp`）：只参考角色的服装组成和全身比例，不复制交叉腿、抬脚、手持手机的姿势，不复制签名。
- 图片 2（`半身.webp`）：作为脸型、蓝色眼睛、红棕色长发、刘海、配色和二次元立绘完成度的主要参考。
- 图片 3（`侧身.png`）：只辅助理解长发、外套和衬衫的侧面层次，不复制饮料瓶和喝饮料动作。
- 没有背面设定，不展示背面，不添加没有参考依据的背面装饰。

## 正向 Prompt

根据三张输入参考图，创建一张可作为后续桌宠动画统一基准的高质量角色全身设定图。角色为成年女性，正面略微三分之二视角，但身体基本朝向镜头。保持参考图中的身份特征：蓝色眼睛、红棕色长直发、整齐刘海、白色衬衫、红色领带、棕色长外套、黑色短裤、深色丝袜和黑色鞋子。

姿势必须是自然、稳定、无动作倾向的标准站姿：身体直立，脊柱自然，肩膀放松且保持水平；双腿自然分开约一个脚掌宽，左右膝盖伸直但不僵硬；两只脚完整可见、处于同一地面高度、脚掌平放并同时承重；禁止交叉腿、踮脚、抬脚、屈膝、迈步、跳跃或悬空。双臂自然垂在身体两侧，双手放松，不叉腰、不抱臂、不触碰脸或头发，不拿任何物品。表情平静、温和，嘴巴自然闭合。

风格为精致、干净、统一的高质量二次元游戏立绘：稳定清晰的线稿，细致的赛璐璐上色，柔和但明确的明暗层次，头发、眼睛、服装褶皱和鞋子具有足够细节。全身从头顶到鞋底完整显示，角色居中，四周留有充足空白，不裁切头发、手指、外套或鞋子。

背景必须是完全均匀的纯绿色 #00ff00，不能有渐变、阴影、纹理、地面线、倒影或环境。角色自身不得使用 #00ff00。不要文字、标志、签名和水印。

## 负面约束

禁止未成年外观、Q 版比例、低清晰度、粗糙线稿、模糊五官、脸部漂移、瞳色变化、发型变化、服装变化、短裙、长裤、换鞋、交叉腿、抬脚、单脚承重、踮脚、屈膝、迈步、奔跑、跳跃、身体倾斜、抱臂、叉腰、手扶头发、手机、饮料瓶、道具、额外人物、额外肢体、手指错误、裁切身体、复杂背景、投影、文字、签名和水印。
